## Root cause and repair accounting

The alerted source was committed at `2026-07-21T13:20:39Z`. Its CI and Rust
runs were created at `13:20:42Z` and `13:20:43Z`, respectively. The first gate
failure was sampled only 25 seconds after the commit and 21–22 seconds after
workflow creation. At that time the CI unit, integration, and PowerShell jobs
and the Rust workspace job were still running. The second sample was 71 seconds
after the commit and 67–68 seconds after workflow creation. The CI unit and
PowerShell jobs had passed by then, while the CI real-libmpv integration job
and the Rust workspace job were still running. CI completed successfully at
`13:21:58Z`, and Rust completed successfully at `13:23:58Z`.

The historical evaluator treated missing, mismatched, queued, or in-progress
exact-SHA workflow evidence as an immediate `source-main-ci` failure. Pull
request #512 changed pending evidence inside a 900-second window to a blocking
warning with reason `source-main-ci-settling`, while preserving immediate
completed failures and fail-closed overdue evidence. Pull request #524
completed the repair by anchoring settling to the matching GitHub `PushEvent`,
falling back to matching workflow creation time, and treating a successful
empty workflow query as pending rather than an API failure. The evaluator and
live collector retain regression coverage for settling, overdue pending
evidence, completed failures, delayed pushes, empty run histories, and missing
event-feed evidence. After #524 landed, Maestro recorded that the configured
runtime outcome was verified in the closure of
[`#508`](https://github.com/BeFeast/ok-player/issues/508#issuecomment-5039482974).

The public evidence proves that #505 was not a completed source-workflow
failure. The generated pipeline context named
`rust/crates/okp-core/src/playback_failure.rs`, but the triggering commit did
not modify that file and the exact-source Rust workflow passed. Editing
playback failure classification would therefore have no demonstrated causal
link to the gate defect.
