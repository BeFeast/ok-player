## Limitations and holds

- The historical rolling manifest URL is mutable by design. The build-175
  manifest digest and immutable versioned package assets bind the accepted
  recovery state recorded here; later candidate promotion may replace only the
  rolling pointer.
- No deliberate new `stale_generation` race was induced against the public
  candidate lane. The cited hosted false green supplies that live evidence;
  deterministic fixtures pin the repaired decision contract without racing or
  mutating production.
- Headless/source verification does not attest playback, GPU, compositor,
  portal, desktop notification delivery, or installed-package behavior. None is
  part of this release-engineering outcome-classification issue.
- Notification transport, GitHub dispatch credentials, and the Maestro project
  pause actuator remain operator-owned private configuration. The repository
  provides the reason code, severity transition, intake health gate, and wiring
  contract; it does not expose or install private infrastructure.
