screenshot states. Eighteen captured states were independently measured; all image redlines passed,
but the buffered-timeline functional rail assertion is a required failure.

interaction regression:

```text

```

The independent keyboard-fullscreen smoke passed. That result shows fullscreen state and rendering
can work through the keyboard path; it does not excuse the required stationary double-click or
timeline failures.
