Marketing/showcase screenshots used by the repo `README.md` and the landing page (`docs/site/`).

Drop the PNGs here with **exactly these filenames** so the existing `<img>` references resolve:

| Filename | What it shows | Source frame |
|---|---|---|
| `chapters-godfather.png` | Chapter panel with per-chapter thumbnails (hero image) | The Godfather (1972), 23 chapters |

Guidelines:
- Capture at the app's default window size (≈1216×688 in these shots) or larger; PNG, no scaling artifacts.
- Prefer frames that are visually rich but not spoiler-heavy.
- Keep titles/paths in shot free of anything private (local user folders, network hosts).
- If you re-shoot at a different size, that's fine — the `width=` attributes in the README/landing scale them down.
